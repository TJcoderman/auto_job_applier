Onboarding Checklist

Follow these steps to prep your agent for production-grade job hustling.

← Back to dashboard

1. Configure identities

  • Copy .env.example → .env and set ADMIN_USERNAME, ADMIN_PASSWORD_HASH, and SECURITY_SESSION_SECRET.
  • Use python -m agentic_job_hunter utils hash-password to generate the hash.
  • Store API keys securely via python -m agentic_job_hunter profile set-credential.

2. Load profile & résumé

  • Populate config/profile.json with contact info and resume path.
  • Upload your base résumé in Markdown/Plaintext for best tailoring results.
  • Add portfolio links (GitHub, LinkedIn, personal site).

3. Tune job discovery

  • Set keywords, locations, and target compensation in .env.
  • Extend agentic_job_hunter/job_scraper with bespoke sources if needed.
  • Use python -m agentic_job_hunter jobs scout to validate search coverage.

4. Tailoring & automation

  • Verify OpenAI access or swap in your preferred LLM provider.
  • Install browser automation assets via playwright install chromium.
  • Test Lever automation in review mode before enabling auto-submit.

5. Run & review

  • Launch the agent via web UI or python -m agentic_job_hunter run --max-jobs 3.
  • Inspect fit scores and recruiter feedback in the dashboard.
  • Log every external response to improve future prioritisation.

6. Scale to production

  • Deploy with Docker: docker compose up -d.
  • Integrate notifications (Slack/Email/SMS) per the deployment checklist.
  • Review the partner playbook for enterprise onboarding & monetisation ideas.
